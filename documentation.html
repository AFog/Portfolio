Inroduction

  • This webpage was built from scratch and no template was used. I added all features by using examples and tutorials from www.w3schools.com, getbootstrap.com, youtube.com and several other websites.
  • To begin I looked for a CV webpages that I liked and wrote down the feature they used that I would like to incorporate into my page
  • The project is hosted on aaronfogarty.com
  • The project is available at GitHub

Dynamic Content

  • I used AngularJS to create some dynamic content
  • I dynamically added my exam results from a file rather than having to manually input the values. This can be seen in the Results JQuery Accordion
  • I also dynamically added the current date and time. Thuis can be seen in the footer of the page.
  • Enabling multiple angular components requires manually bootstrapping each of them using the angular.bootstrap command

  • I used googe maps api to show my home adress in the contact section of the page. This required creating a google project enabling Maps JavaScript access and getting an API key to connect

Interactive Elements

  • I found many elements to use at W3schools and getbootstrap
  • I used a parallax to create the page heading that has my name, this is a nice feature and can be seen on a lot of the nicest web CVs.I aslo use a BS jumbotron in the header to show my photo and title
  • The image in the jumbotron is a Modal and enlarges the image for closer view
  • I aso used a Modal for one of my contact forms. This can be seen in the about section beside the social media images. This is a functional form, any messages submitted are sent to my email address. This feature was possible using 'JotForm'
  • For the whole about section I used a carousel to display differnet sections of my CV. I seen this feature and like it and built my own version into the page
  • I also used carousels in the Projects section to display project images
  • The CV is built for multiple devices and works on mobile, I used a Grid system to enable this
  • The Navnbar menu changes to a dropdown menu in mobile devices
  • Navbar Toggle was used to replace the navbar with a button with arrows, on smaller screens
